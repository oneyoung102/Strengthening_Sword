# Strengthening_Sword

> 본 파일은 `<windows.h>` 라이브러리를 사용하여 window 운영체제에서만 작동합니다.

> 고등학교 2학년 겨울방학 때, 취미로 만든 검강화하기 게임입니다.

<hr/>

## 게임 방법
  > **검강화하기.cpp**를 실행시켜 터미널에서 플레이합니다.   
  > **text.txt**는 플레이어가 강화했던 검의 정보를 담고 있습니다.
> 
  - #### 메뉴
    - Strengthen   
      검을 강화할 수 있습니다.   
      검을 강화할 때마다 검의 외형이 변화합니다.
      - Power   
          검의 기본 공격력을 증가시킵니다.   
      - Hardness
          검의 데미지 변동 폭을 줄여주며, 최대 데미지와 평균 데미지는 증가시킵니다.
      - Sharpness   
          검의 치명타 확률, 데미지를 증가시킵니다.
    - Swords   
      자신의 검과 다른 사람의 검의 외형을 확인할 수 있습니다.   
      허수아비를 공격하여 데미지를 확인할 수 있습니다.   
    - probability   
      단계별 강화 확률을 보여줍니다.   
      강화 성공, 실패, 파괴 확률이 존재합니다.
